# Сайт Лебедин Виталий

Это персональный проект сайта школы‑лицея в селе Кордай.  
Сайт сделан с помощью ChatGPT и содержит:

- Главную страницу с карточкой Лебедина Виталия и кнопками:
  - Telegram
  - WhatsApp
  - Tic-Tac-Toe
  - Мины
  - Вдохновитель (биография учителя)
- Страницу с биографией учителя физики и списком класса (`teacher.html`)
- Игру "Крестики-нолики" против бота (`ttt.html`)
- Игру "Мины" (`mines.html`)
- Биографию учителя информатики (`boa.html`)
